\doxysubsection{Detailed Description}
Represents a task in a Unix-\/based system. 

Implements the {\ttfamily \mbox{\hyperlink{classGeneralTask}{General\+Task}}} interface and provides additional functionality for managing task states, priorities, and attributes.
